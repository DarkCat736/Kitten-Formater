Kitten Formater

Welcome to Kitten Formater! This is the first software ever released by the young developers at Kitten's Tech!
Currently we are at version 1.0.0, so if there is any problem PLEASE let us know. In this helpful program, all you
have to do to format a folder or drive on your computer is choose it and hit the format button! Run the installer
and format away! Enjoy the program and keep an eye out for updates!
